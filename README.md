# Agrognom (Backend)

The backend part of my project for farmers, which allows them to receive
recommendations using Gemini AI based on their own data.

</div>

---

## Overview

This repository contains the server side of Agrognom application, written on Kotlin Spring Boot.
The system provides an API for all basic functions of the application like:
- Registration/Authorization;
- Fields managing;
- Receiving recommendations;
- Getting current weather;

---

## Getting Started

### Prerequisites

Before starting, make sure you have installed:

- **JDK 17+**
- **Gradle 8+**
- **MySQL** (or other DBMS)

### Installation

Build agrognom_Backend.git from the source and intsall dependencies:

1. **Clone the repository:**

    ```sh
    ❯ git clone https://github.com/xasanit/agrognom_Backend.git
    ```

2. **Navigate to the project directory:**

    ```sh
    ❯ cd agrognom_Backend.git
    ```

3. **Install the dependencies:**
All dependencies are in build.gradle.kts file.
Gradle will download them automatically.

### Usage

Run the project with:

**Using [gradle](https://kotlinlang.org/):**
```sh
  gradle run
```
